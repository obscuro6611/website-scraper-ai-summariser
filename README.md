# Website Scraper + AI Summariser

A Python application that extracts content from websites, processes and validates the data, generates summaries, and exports results to structured formats such as JSON and Markdown.

## Features

- Website content extraction
- Content parsing and validation
- AI-ready summarisation pipeline
- JSON export
- Markdown export
- GUI interface
- Executable build support

## Technologies

- Python
- Requests / HTTP clients
- Content parsing
- JSON
- Markdown
- PyInstaller
